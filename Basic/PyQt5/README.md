### basic

---

- PyQt5는 Qt5 어플리케이션 프레임워크에 대한 파이썬 버전
- Qt는 플랫폼에 관계없이 다양한 기능을 포함하는 C++ 라이브러리이자 개발툴
- 필요 설치 : 
  - PyCharm : 파이썬 개발에 가장 널리 사용되는 통합 개발 환경(IDE) - 나는 VsCode
  - Anaconda
