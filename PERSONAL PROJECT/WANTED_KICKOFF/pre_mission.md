# 프리 미션



### 경력 프로젝트

- #### 한국전자기술연구원(2020.8. - 2024.3. 현재)

  - ##### 5G 기반 선제적 위험대응을 위한 예측적 영상보안 기술 개발 연구에 백엔드 개발자로 참여

    - Fact
      - Python Flask 기반의 API 서버와 kubernetes를 연동시켜 웹 서비스에서 노드를 클러스터링하고 AI 소프트웨어를 배포가능하게 함
      - 마이크로 서비스를 제공하기 위해 AI 소프트웨어를 경량화 시켜 각 노드에 배포시키고 협업을 하여, 각 노드의 부화를 감소시켜 클러스터의 성능을 향상시킴
      - Prometheus의 데이터를 기반으로한 시각화 툴인 Grafana를 웹 서비스를 통해 배포하고 연동하여 클러스터의 상태를 확인하고 안정화시킴
      - 1개의 5G기반 실시간 CCTV를 제작하여 영상데이터를 활용하고, 5개의 차량용 CCTV 영상데이터를 GPS, Gyro 센서의 메타데이터를 활용해 오산시 스마트시티 통합운영센터에 설치하고 영상 보안 데이터를 제공
        - 링크 : https://www.asiatoday.co.kr/view.php?key=20231228010017226
    - Lesson
      - 실시간 영상 데이터를 활용하는데 네트워크, 성능의 과부하 문제가 있는데 kubernetes를 이용해 과부하를 감소시켜 성능을 향상시킬 수 있었음
      - kubernetes와 Python Flask 웹 서비스를 연동하는데 세세한 환경설정이 필요함을 인지하여 웹 서비스에서 환경을 입력할 수 있게 하여 문제를 해결함

  - ##### 논문

    - (주) 쿠버네티스 기반 에지 클라우드 환경에서  실시간 영상 스트리밍 분산 처리 기술
    - (부) 5G 기반 영상보안 시스템 프레임워크

  - ##### 특허

    - 국내3/국제2 특허 출원
      - PTZ CCTV 카메라 자동제어 방법 및 시스템(국제)
      - 반자동 CCTV 영상 검색을 위한 전자장치 및 방법(국제)
      - 엣지 CCTV를 위한 엣지 AI 모델 생성 방법 및 시스템

  - ##### 소프트웨어 등록

    - 모빌리티 단말의 실시간 영상 및 위치 정보 저장 소프트웨어
    - 모빌리티 단말의 실시간 영상과 위치 정보에 대한 전송 및 재전송 소프트웨어

  - ##### 시제품

    - 영상보안 서비스용 엣지서버 관리기술
    - 5G CCTV
    - 5G 단말 영상의 실시간 전송 및 재전송 기술



### 교육 프로젝트

- #### 멀티캠퍼스

  - 프로젝트 1 (Java) - 골목상권
    - Fact
      - Leaflet을 사용해 서울 지도에 최상위 상권 15개를 마커로 표시하고 해당 상권에 대한 정보 제공 후 해당 페이지와 연결
      - 해당 상권으로 들어가 원하는 업종 선택 하면 업종에 관한 정보를 저장된 데이터 베이스 내 정보들을 갖고 와서  AJAX를 사용해 실시간으로 출력
      - d3.js를 활용해 매출액을 분석해주고 연령별, 성별 매출 금액 분석 그래프를 구현하여 메인페이지 및  분석 페이지에 제공
    - Lesson
      - Java Spring MVC 로 개발한 첫 프로젝트로 R Selenium, Leaflet, d3.js 등 다양한 기술을 활용하여 웹서비스를 제작하여 서비스 개발에 대한 열정을 갖게됨
  - 프로젝트 2 (Java) - pmkim
    - Fact
      - 각 편의점명으로 검색된 인스타그램 최신 게시물 각 10,000개의  데이터 크롤링, 편의점 상품 데이터 크롤링
      - 최신 5,000개의 게시물에 대한 감정분석 실시, 감정분석 결과 그래 프 제공 및 가장 높은 긍정표를 받은 편의점 순으로 편의점 랭킹 제공(SentimentalAnalysis 활용)
      - 각 10,000개의 데이터를 형태소 분석하여 추출된 명사를 워드클라 우드로 표현하여 가장 핫한 아이템 정보를 제공(Rmecab 활용)
      - 최하단에 각 편의점의 상위 아이템 정보 및 인스타그램 연동 
    - Lesson
      - 팀원간의 마찰으로 많은 기능들을 완벽하게 구해내지 못함
      - 프로젝트 중간에 삭제하고 추가할 점들이 있다는 점을 알게됐고, 최소 필요한 기능들을 먼저 개발하는 방법을 택하게됨
  - 프로젝트 3 (Python) - i.hub
    - Fact
      - DB를 Aquery Tool을 사용해 정의
      - 서울시 열린 데이터광장에서 제공하는 조회수, 다운로 드수 별 API 랭킹을 크롤링하여 AJAX 기술을 활용하 여 실시간으로 본 사이트에 랭킹 자료제공
      - XML, json형식으로 제공되는 API들의 반환값들을  json 형태로 변환 혹은 저장하여 Django의 media  폴더에 저장하여 다운로드 기능 개발
    - Lesson
      - 첫 Python 프로젝트로 사용자에게 실시간으로 서비스를 제공하는 기능을 개발해봄
      - 팀원들과 함께 프로젝트를 수행하는데 있어서 자율적으로 학습을 하면서 하게되는 계기가 됨
